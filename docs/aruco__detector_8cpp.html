Author
Ayan Mazhitov
Version
1.0
Date

Description :

Node to detect markers around the robot by moving the camera. It request to the marker_server node to know the markrer meaning, and publishes the received informations to the nodes /markerIDs and tells others nodes that it has finished its execution by publish to a topic /isDone.

Main function for the aruco_detector node.

This function creates an instance of the ArucoDetector class. It then enters a loop where it sends camera movement commands and spins the ROS event loop until all markers have been detected.
